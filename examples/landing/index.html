Keyboard & accessibility

            

              Drag-and-drop is notoriously mouse-only. This engine makes reordering fully operable
              from the keyboard, with live-region announcements — try it on the recipe demo above.
            

          


          

            

              
                Reordering
              
              
                

                  Tab
                  Focus a list item
                

                

                  Space
                  Grab the focused item (and drop it again)
                

                

                  ↑ ↓
                  Move the grabbed item up / down
                

                

                  Esc
                  Cancel the move and return it home
                

              
            


            

              
                Multi-select
              
              
                

                  Ctrl/⌘ + click
                  Add or remove a row from the selection
                

                

                  Shift + click
                  Select a contiguous range
                

                

                  drag
                  Move every selected row together as one batch
                

                

                  aria-live
                  Each grab, move and drop is announced
                

              
            

          

        

      


      
      

        

          

            
Why this one

          

          

            

              
Zero dependencies

              

                The core pulls in nothing. It's ~10 kB gzipped of plain TypeScript over the Pointer
                Events API — no jQuery, no framework, no polyfills.
              

            

            

              
One engine, real adapters

              

                All the behaviour lives in dnd-core. Each adapter is a thin idiomatic
                binding — <Draggable> with v-model in Vue,
                controlled items in React, use:draggable in Svelte, a primitive in
                Solid.
              

            

            

              
Pointer Events, everywhere

              

                One code path for mouse, touch and pen — with a drag threshold, touch delay and
                autoscroll — so it feels right on a trackpad and a phone alike.
              

            

            

              
Keyboard-accessible

              

                Grab, move, drop and cancel from the keyboard, with aria-grabbed,
                aria-posinset and live-region announcements built in — not bolted on.
              

            

            

              
Groups, clone & swap

              

                Cross-list dragging with pull/put rules, clone-on-drag
                palettes, swap mode for grids, spill-to-delete and nested trees — all in the core.
              

            

            

              
FLIP animation

              

                Every reorder animates smoothly from its old position to its new one, and honours
                prefers-reduced-motion without you writing a keyframe.
