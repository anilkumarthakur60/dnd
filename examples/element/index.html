    10 Nested lists (trees)

    Trees are just lists inside list items — recurse a <dnd-list> inside each <li> and give every level the same group attribute so items drag freely between branches and depths.

    Use it for file trees, comment threads, org charts.

    Drag any row into another branch — the JS tree model stays in sync.

    11 Programmatic API & live events

    Reach the engine via el.sortable.list; every event re-dispatches as a dnd-* CustomEvent.

    Use it for "add row" buttons, undo/redo, server sync.
